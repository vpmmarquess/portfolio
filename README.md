# Portfólio Pessoal

Portfólio desenvolvido com HTML e CSS para apresentação profissional, incluindo informações sobre mim, habilidades, projetos e formas de contato.

## 🚀 Funcionalidades

- Apresentação profissional
- Seção "Sobre mim"
- Listagem de habilidades
- Exibição de projetos
- Informações de contato
- Layout responsivo

## 💻 Tecnologias utilizadas

- HTML
- CSS

## 🌐 Acesse o projeto

https://vpmmarquess.github.io/portfolio

## 📌 Objetivo do projeto

Este projeto foi desenvolvido para apresentar meu perfil profissional, habilidades e projetos, com foco em oportunidades na área de desenvolvimento de sistemas.
